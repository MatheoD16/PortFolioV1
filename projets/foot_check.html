Football-Checker


Une application de gestion de championnat de foot en Java

Football-Checker



Caractéristiques du projet :


  • Java/JavaFx
  • Projet personnel
  • En cours ⚙️
  • GitHub: Work in progress

Description :


1-Pourquoi ce projet ?

J'ai eu l'idée de ce projet car quelqu'un dans ma famille joue au foot et il ne peut pas voir le classement de son championnat. J'ai donc voulu faire cette application pour qu'il puisse rentrer les données de son championnat pour avoir accès à un classement calculé avec le système de point officiel.

2-L'application

Cette application permet à l'utilisateur de rentrer des noms d'équipes pour son championnat, puis une fois les équipes validées, de rentrer le résultat des matchs pour pouvoir visualiser le classement du championnat. Je compte aussi utiliser une base de données pour stocker les championnats de l'utilisateur.

⚠️ Cette application est actuellement en cours de développement mais il est déjà possible d'ajouter les équipes et de naviguer dans les différentes fenêtres de l'application !

J'ai donc pour l'instant fait les différentes classes pour gérer l'ajout des équipes ainsi que les différentes vues qui seront utiles pour l'application. J'ai aussi implémenté l'architecture MVC pour ce projet en utilisant 1 controlleur pour chaque vue et en utilisant une classe pour gérer les différentes vues.
