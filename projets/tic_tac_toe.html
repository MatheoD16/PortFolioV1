Tic-Tac-Toe


Un jeu de morpion réalisé en Java

Caractéristiques du projet :


  • Java/JavaFx
  • Projet personnel
  • Terminé ☑️
  • GitHub: Tic-Tac-Toe

Description :


1-Pourquoi ce projet ?

J'ai eu l'idée de faire ce projet après mes premiers cours de programmation d'IHM. J'avais envie d'utiliser ce que je venais juste d'apprendre dans un vrai projet, j'ai donc fais cette petite application en quelques jours. C'est aussi mon premier projet personnel.

2-L'application

Cette application permet à l'utilisateur de jouer au morpion avec un autre joueur (en utilisant le même ordinateur).

J'ai donc implémenté plusieurs classes pour gérer la grille de jeu et les cases. J'ai aussi utilisé l'architecture MVC pour ce projet.

Il est donc possible de démarer une partie en cliquant sur une case de la grille pour la changer de couleur, le programme gère les différents cas de défaite ou de victoire.

Quand la partie est terminée, il n'est plus possible de cliquer sur les cases et il faut cliquer sur le bouton recommencer pour démarer une nouvelle partie.
