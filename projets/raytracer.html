RayTracer


Une librairie Java de génération d'images

RayTracer



Caractéristiques du projet :


  • Java
  • Projet en groupe de 4
  • Terminé ☑️
  • Réalisé en 5 jours

Description :


1-Pourquoi ce projet ?

Dans le cadre de mes études, il m'a été demandé de programmer en 5 jours, une librairie Java pour générer des images, pour cela j'ai collaboré avec 3 autres personnes. Nous nous sommes répartis les différentes tâches tout au long de la semaine.

2-La librairie

Cette librairie permet de générer des images à l'aide d'instructions dans un fichier .txt.

Cette librairie peut générer des images en prenant en compte la gestion de la lumière et des reflets.

Pour cela il est possible d'ajouter une scène, plusieurs lumières et même différents objets créés à l'aide de formes basiques comme des ronds ou des triangles.
